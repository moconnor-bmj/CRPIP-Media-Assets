WCAG 2.0 Level AA Accessibility Audit of Openheart JCore Test Site

Report Creator: Michael OConnor, January 27, 2015

Evaluation Commissioner: Helen King

Summary of the evaluation findings

Undertaken as part of the Journal Network Project and the migration of BMJ Journals to a new Highwire JCore platform.

Scope of the evaluation

Website name Openheart Journal (JCore Test site):
Scope of the website

Site: http://jnl-bmjopen.drupal-stage-jnl-web02.highwire.org/
Pages: Home; Article; Archive; Latest Content; Search; Advanced Search; Table of Contents.
Conformance target Level AA
Additional evaluation requirements

The report will include a list of all errors identified by the Evaluator.
The report will include a description of the problem and repair suggestions for any errors listed.
The evaluation will cover a selected sample of web pages and web content of the website.
